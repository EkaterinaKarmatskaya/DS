Задание
Цель задания - проанализировать удержание пользователей (Retention Rate) в приложении. Вам необходимо составить SQL-запрос для расчета еженедельного Retention Rate на основе предоставленных данных.

Ваш запрос должен возвращать результат со следующими полями:

week: номер недели с момента регистрации пользователя (начинаем с 1 недели, нулевую пропускаем).
active_users: количество активных пользователей на этой неделе
total_users: всего пользователей зарегистрированных к этой неделе
retention_rate: процент удержания пользователей на данной неделе.
